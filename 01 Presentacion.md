# GitHub Copilot Dev Days | Ciudad del Este

## 🎯 Qué es GitHub Copilot Dev Days (la versión corta y precisa)

**GitHub Copilot Dev Days** es una **iniciativa global, presencial y liderada por la comunidad**, impulsada desde GitHub y Microsoft, que se realiza entre el **15 de marzo y el 15 de mayo de 2026** en ciudades de todo el mundo.

El objetivo es que desarrolladores, entusiastas y comunidades tecnológicas locales **se junten cara a cara** para explorar el poder de **GitHub Copilot** a través de experiencias **prácticas y manos en el código**.

Cada evento local dura entre **3 y 9 horas** según el formato, y combina:

1. **Sesiones técnicas** — charlas y demos sobre GitHub Copilot.
2. **Workshops hands-on** — laboratorios guiados donde los asistentes usan Copilot en escenarios reales.

**Nuestra edición de Ciudad del Este es un evento oficial autorizado, avalado por el programa global de GitHub.**

Les cuento brevemente qué es este evento. **GitHub Copilot Dev Days** es una iniciativa **global** que se está realizando en decenas de ciudades del mundo entre marzo y mayo de este año. Es un programa oficial de **GitHub**, diseñado para que las comunidades locales se junten de forma presencial a aprender, compartir y poner las manos sobre el código usando **inteligencia artificial aplicada al desarrollo de software**.

Hoy somos parte de eso. Somos **una de las sedes oficiales** de esta iniciativa global, y lo hacemos **desde Paraguay, desde Ciudad del Este**, con una comunidad que está creciendo fuerte.

Este evento es **100% gratuito**, y lo es porque hay dos pilares que lo hacen posible:

- **Mi LAB – Hub de Innovación Colaborativo**, que nos cedió este espacio sin costo. Un aplauso para ellos.
- Y el trabajo **completamente voluntario** de organización, que se hizo pro bono para la comunidad.

- [ ] Presentarse como organizador (Michael Sahlmann).

Lo que vamos a vivir hoy son **9 horas de contenido**: **sesiones técnicas**, un **laboratorio práctico**, un **taller hands-on**, espacio para **preguntas**, **networking**, **sorteos** al final del día, y dos momentos de **coffee break** para recargar energías.

Tres cosas importantes antes de arrancar:

1. **Este evento es para todos los niveles.** Si nunca tocaste IA ni GitHub Copilot, estás en el lugar correcto. Si ya sos desarrollador experimentado, también.
2. **Para el taller práctico necesitan su notebook personal con cargador.** Si la olvidaron cargada, ahora es un buen momento para enchufarla.
3. **Existe un Código de Conducta** que pedimos respetar: este es un espacio de aprendizaje, inclusivo y respetuoso para todos.

Ahora sí, **bienvenidos**, pónganse cómodos, y empecemos.
